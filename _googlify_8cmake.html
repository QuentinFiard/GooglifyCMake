get_output_file ( TARGET  ,
OUT   
)

Returns the absolute path to the output file of a target. For executable targets this is the path to the executable binary. For library targets this is the path to the library file. For other targets, returns the value of the TARGET_FILE target property.

Warning
This issues CMake generator expressions as it is in general not possible to get the name of the output file of a target if it has not already been processed by CMake. Generator expressions postpones the evaluation of the output file path.
Parameters
TARGETAbsolute name of a target.
OUTName of a variable to save the result in.
